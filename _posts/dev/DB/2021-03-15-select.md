## 기본 문법
```sql
SELECT 필드명 
FROM 테이블명
WHERE 조건
ORDER BY 정렬할 필드명 ASC/DESC
LIMIT 표시할 상위 레코드 수
```

## 문제 예시
---

**1. 동물 보호소에 가장 먼저 들어온 동물의 이름을 조회**

```sql
SELECT NAME FROM ANIMAL_INS
ORDER BY DATETIME ASC
LIMIT 1
```
